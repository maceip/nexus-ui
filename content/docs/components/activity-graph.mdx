A GitHub-style contribution heatmap for activity data. It also supports a loading mode that animates the grid like a side-scrolling game loop, so it can double as a spinner or skeleton for repository views.

## Usage

```tsx keepBackground
import { ActivityGraph } from "@/components/nexus-ui/activity-graph";
```

```tsx keepBackground
<ActivityGraph
  data={[
    { date: "2026-03-11", count: 5 },
    { date: "2026-03-12", count: 1 },
  ]}
/>
```

## Loading mode

Set **`loading`** to animate the graph as a moving loading surface. This keeps the same prop surface as the activity graph while making it useful as a spinner-like placeholder.

```tsx keepBackground
<ActivityGraph data={[]} loading loadingSpeed={0.9} weeks={20} />
```
