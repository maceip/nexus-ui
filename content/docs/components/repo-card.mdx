Display GitHub repository metadata in a compact card that matches Nexus UI styling. `RepoCard` supports build-time fetching, pre-fetched data, and visibility toggles for language, license, updated date, and topic tags.

## Usage

```tsx keepBackground
import { RepoCard } from "@/components/nexus-ui/repo-card";
```

```tsx keepBackground
<RepoCard owner="shadcn-ui" repo="ui" />
```

Async server component. By default it fetches the GitHub API and caches the result for one hour with Next.js revalidation. Pass `data` to skip the fetch and render pre-fetched repository details instead.

## Notes

- `RepoCard` is an async server component and is safe to render directly in app-router server pages.
- The optional `GITHUB_TOKEN` environment variable raises GitHub API rate limits for server-side fetches.
- Archived repositories render a warning badge; forked repositories render a fork badge.
- Unknown languages fall back to a neutral dot color.
