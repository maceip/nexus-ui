## Usage

```tsx keepBackground
import { CommitGraph, type Commit } from "@/components/nexus-ui/commit-graph";
```

```tsx keepBackground
<CommitGraph commits={commits} />
```

Client component. Pass commits in topological order (newest first) with `parents` hashes. Each row opens a popover with commit details.

## API Reference

### CommitGraph

<TypeTable
  type={{
    commits: {
      type: "Commit[]",
